Dynamic Flame Particle Art

This interactive web art piece showcases a dynamic flame particle system that responds to user input. As the user moves the mouse, vibrant particles are generated, creating a mesmerizing visual experience. Expanding Quadrilateral Animation

This captivating web animation showcases an expanding quadrilateral that subdivides into four smaller quadrilaterals at its corners. The smooth transformation is achieved using CSS keyframe animations, resulting in an eye-catching effect that alternates between the main quadrilateral and its subdivisions. The simple yet engaging design demonstrates a creative use of CSS to build interactive and visually appealing experiences.

Interactive Artistic Waving Dots with Gravity and Collisions

This interactive web page features a series of colorful, randomly sized and positioned dots that move according to gravity and bounce upon collision with each other or the window borders. The user can click and drag any dot to move it around the screen. When two dots collide, they change colors and exchange their velocities. The playful and engaging design provides an artistic and dynamic visual experience that demonstrates the creative potential of JavaScript and CSS for user interaction and animation.
